# Лабораторная работа №1: Работа с Git и GitHub

Проект представляет собой приложение для расчета параметров геометрических фигур (прямоугольник, треугольник и окружность).

## Функциональные возможности
Программа позволяет производить следующие вычисления:
1. Прямоугольник: нахождение площади и периметра по двум сторонам.
2. Треугольник: расчет параметров по заданным основаниям и высоте (или сторонам).
3. Окружность: нахождение площади круга и длины окружности по заданному радиусу.

## Структура проекта
Исходный код разделен на модули для раздельной компиляции:
* main.cpp — реализация пользовательского интерфейса и логики выбора фигур.
* rectangle.h / rectangle.cpp — описание и реализация функций для прямоугольника.
* triangle.h / triangle.cpp — описание и реализация функций для треугольника.
* circle.h / circle.cpp — описание и реализация функций для окружности.

## Порядок сборки и запуска
Для компиляции проекта необходимо использовать компилятор g++:

1. Клонирование репозитория:
git clone git@github.com:erdeniha/Laboratory1.git

2. Компиляция всех модулей:
g++ main.cpp rectangle.cpp triangle.cpp circle.cpp -o main

3. Запуск исполняемого файла:
./main

## Работа с системой контроля версий
Разработка велась с использованием ветвления Git:
* Ветка main — стабильная версия кода.
* Ветки rectangle, triangle и circle — поэтапная реализация функционала участниками команды.
